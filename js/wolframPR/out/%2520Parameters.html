new - Parameters(height, width, XLocation, YLocation, color, border)

Basic constructor for creating a cell object with specific dimensions, location, color, and border. This constructor is used to define the properties of individual cells in a grid, such as their size, position on the grid, color (alive or dead), and border color. The cell can represent a state in a simulation or grid-based system.
Parameters:
Name Type Description
height number The height of the cell.
width number The width of the cell.
XLocation number The X-axis location of the cell.
YLocation number The Y-axis location of the cell.
color string The color of the cell (e.g., 'black' or 'white').
border string The color of the border around the cell.

new - Parameters(message, color, canvas)

Constructor for creating a log message object that can be displayed on a canvas. This constructor sets up the message, its color, and the canvas on which it will be rendered.
Parameters:
Name Type Description
message string The message text to be displayed in the log.
color string The color of the message text (e.g., 'red', 'green').
canvas HTMLCanvasElement The canvas element where the message will be displayed.
